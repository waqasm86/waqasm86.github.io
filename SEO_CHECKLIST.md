## Common Issues & Fixes

### Issue: Search Console not indexing pages
**Fix:**
1. Submit sitemap.xml
2. Request indexing via URL Inspection
3. Check robots.txt isn't blocking
4. Verify no noindex tags

### Issue: Low CTR in search results
**Fix:**
1. Improve title tags (add power words)
2. Enhance meta descriptions (include CTA)
3. Add structured data for rich snippets
4. Test with different title variations

### Issue: Slow page load
**Fix:**
1. Compress images (WebP format)
2. Enable browser caching
3. Minify CSS/JS
4. Use CDN for assets
5. Lazy load images

### Issue: No backlinks
**Fix:**
1. Create shareable content (tutorials, infographics)
2. Reach out to bloggers for mentions
3. Submit to directories
4. Guest post on relevant sites
5. Create tools others will link to

### Issue: Keywords not ranking
**Fix:**
1. Increase content depth
2. Add more internal links
3. Get quality backlinks
4. Improve on-page SEO
5. Create supporting content
