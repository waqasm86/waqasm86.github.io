    Reporting Bugs

    When reporting bugs, please include:

    System Information: 

    # Python version
python3 --version

# llcuda version
python3 -c "import llcuda; print(llcuda.__version__)"

# GPU information
nvidia-smi

# OS information
cat /etc/os-release

    Error Details: - Full error message - Steps to reproduce - Expected vs actual behavior - Minimal code example

    Example Bug Report: 

    **Title**: CUDA out of memory with Gemma 2 2B on GTX 1050

**System**:
- llcuda version: 0.1.5
- Python: 3.10.12
- GPU: GeForce GTX 1050 (2GB VRAM)
- OS: Ubuntu 22.04

**Issue**:
Getting CUDA OOM error when loading Gemma 2 2B model.

**Code**:
from llcuda import LLM
llm = LLM(model="gemma-2-2b-it")  # Fails here

**Error**:
RuntimeError: CUDA out of memory...

**Expected**: Should work on 2GB VRAM
**Actual**: OOM error
